# 🛍️ ShopEase - Modern E-Commerce Website

## 📖 About the Project

ShopEase is a modern, responsive, and user-friendly e-commerce website developed using **HTML5, CSS3, and Vanilla JavaScript (ES6)**. The project focuses on creating a clean shopping experience with responsive layouts, interactive UI components, and a fully functional front-end.

## ✨ Features

* 🏠 Responsive Home Page
* 🛒 Shopping Cart
* 💳 Checkout Page
* 🔍 Product Search
* 📂 Product Categories
* 📄 Product Details Page
* 👤 Login & Registration Page
* 📞 Contact Page
* ℹ️ About Page
* 📱 Mobile-Friendly Design
* ⚡ Fast and Lightweight
* 🎨 Modern UI/UX

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
